==================== SAE J1939 notes ==================

# J1939 protocol : ISO 11898 - 2 / CAN 2.0B
SAE J1939 is a set of standards that define how ECUs communicate 
via the CAN bus in *heavy-duty vehicles*

J1939 provides a higher layer protocol (HLP) based on CAN 
as the "physical layer"
