# diagnostics protocols

1/ On board diagnostic : Monitors Engine ECU 

Warns the users if a malfunction is detected when engine running/stops

- OBD
- OBD II (ISO 9141/SAE J1978)

2/ Off board diagnostic : Monitors All (others) ECU in the vehicule

Warning and store the error code(fault) to be retrieved in the garage
 
- UDS (ISO 14229)
--> UDS on CAN (ISO 14229-3)
--> UDS on IP (ISO 14229)
- KWP2000
- diagnostics over K-Line
- SAE-J1939/73
